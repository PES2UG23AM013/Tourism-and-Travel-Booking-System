SQL Query:
SELECT c.Cname, COUNT(td.DependentName) AS Total_Dependents FROM Customer c LEFT JOIN TravelDependent td ON c.CustomerID = td.CustomerID GROUP BY c.CustomerID;

Description: This query lists all customers along with their total number of dependents using a LEFT JOIN and GROUP BY.

SELECT PackageName, PackagePrice FROM TourPackage ORDER BY PackagePrice DESC LIMIT 3;

Description: This query retrieves the top 3 most expensive tour packages, ordered by price in descending order.

SQL Query:
SELECT b.BookingID, c.Cname, h.HotelName, h.Rating FROM Booking b JOIN Customer c ON b.CustomerID = c.CustomerID JOIN Itinerary i ON b.BookingID = i.BookingID JOIN Hotel h ON i.HotelID = h.HotelID WHERE b.Status = 'Confirmed' OR b.Status = 'Paid';

Description: This query shows all confirmed or paid bookings, including customer names, hotel names, and hotel ratings by joining multiple tables.
