About Procedures & Functions
Stored Procedure: calculate_package_total_cost

This procedure calculates the total cost of a package for a given package ID, including package price, hotel price, and transport price.

  • Input: Package ID
  • Output: Package details with estimated total cost
User-Defined Function: TotalAmountSpent

This function calculates the total amount spent by a customer across all their payments.

  • Input: Customer ID
  • Output: Total spent amount in rupees
